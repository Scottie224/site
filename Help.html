Help

To view the trails all you have to do is follow the simple navigation to the trail page that will let you select the trail you want to follow. On the trail page all of the destinations are listed for you to select and to view a basic description, an address and a gallery for the destinations.

The map shows you the first destination of every location, all you need to do is select the marker to find out the location and it will provide a link to the trail that starts there.

If you want to see more information about the trails simply visit the trail websites page and follow the links to each trail website.

Please note that while interacting with phone numbers on android devices will access the phonecall capabilities of the device this does not work on iOS devices.
